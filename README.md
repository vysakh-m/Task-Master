# Task-Master
HackerEarth StackHack Task Management Project . 
This project was implemented using MERN Stack

## Installing
Install server-side node modules
```
npm install
```
Navigate to client directory
```
cd client
```
Install client-side node modules
```
npm install
```

Return back to server directory
```
cd ..
```

### Specifying Keys
Navigate to /config create keys_dev.js
```
module.exports = {
  mongo: YOUR_MONGO_URI_HERE,
  passport_key : YOUR_RANDOM_KEY_HERE
};
```



### Run the project
```
npm run dev
```
